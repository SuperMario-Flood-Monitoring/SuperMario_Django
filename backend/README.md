# 백엔드 디렉터리

이 폴더는 SuperMario SWMM Django 백엔드의 실제 Django 프로젝트 루트입니다.

## 주요 위치

| 경로 | 내용 |
| --- | --- |
| `apps/` | Django app 구현 |
| `config/` | settings, URL, ASGI 설정 |
| `swmm_engine/` | SWMM 변환, runtime, risk, LLM dispatcher |
| `docs/` | 현재 구현 기준 문서 |
| `manage.py` | Django 관리 명령 진입점 |
